
# note

A CLI for managing markdown note files.


## Features

- ❌ Search
- ❌ Editor
- ❌ Backups
- ❌ Web Frontend
- ❌ GUI


## Usage/Examples

```shell
# Opens week's note file in $EDITOR.
# If it's the 22nd week of 2021, the file would be located by
# default at $HOME/note/2021/22.md
$ note
$
$ cat $HOME/note/2021/22.md
```
